## Overview

This is a forked Microsoft educational curriculum for the Model Context Protocol (MCP), aligned with **MCP Specification 2025-11-25**. It contains documentation and runnable code samples across C#, Java, JavaScript, TypeScript, Python, and Rust.

## Architecture

**11 sequential modules (00–11):**
- `00-Introduction` → `01-CoreConcepts` → `02-Security` → `03-GettingStarted` → `04-PracticalImplementation` → `05-AdvancedTopics` → `06-09` (community, best practices, case studies) → `10-StreamliningAIWorkflows` → `11-MCPServerHandsOnLabs`

Each module follows:
```
<module>/
├── README.md          # curriculum content (English source — edit here)
├── samples/           # language-specific code examples
└── solution/          # complete working implementations
```

**Translation system:** 48+ languages auto-generated by GitHub Actions (`co-op-translator.yml`). Never edit files in `translations/` or `translated_images/` directly — only edit English markdown in the root module directories.

## Running samples

Each sample has its own setup. Navigate into the sample directory and follow its README.

```bash
# TypeScript/JavaScript
cd <sample-dir> && npm install && npm start

# Python
cd <sample-dir> && pip install -r requirements.txt && python main.py

# Java
cd <sample-dir> && mvn clean install && mvn spring-boot:run
```

Key sample locations:
- `03-GettingStarted/samples/` — first server and client implementations
- `11-MCPServerHandsOnLabs/` — PostgreSQL + pgvector database integration labs (13 sequential labs)
